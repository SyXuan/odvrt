# Odvrt

Odvrt 全名為 Old driver video rename tool 老司機影片改名工具，能批次更改含有番號的檔案名稱並下載同名縮圖

# 安裝 Install

把zip整包下載下來

安裝Python 3.9

打開 Termianl 輸入 python qt_view.py

# 使用方法 How to use
老司機應該都會吧，把檔案拉進去點執行就可以了，有任何問題可以開issue給我，平日要上班不一定會馬上回
